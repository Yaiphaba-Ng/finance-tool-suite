# Practice App - React Native Learning (Expo SDK 54)

This project is a practice application designed to help learn React Native using Expo SDK 54.

## Project Configuration
- **Expo SDK:** 54.0.33
- **React Native:** 0.81.5
- **React:** 19.1.0
- **Language:** TypeScript
- **Navigation:** Expo Router (File-based)

## Established Conventions
- **Routing:** All screens are located in the `app/` directory.
- **Components:** Reusable UI components should be placed in `components/`.
- **Theming:** Use `components/Themed.tsx` for light/dark mode support.
- **Icons:** Use `@expo/vector-icons` (FontAwesome is currently used).

## Learning Progress
- [x] Project initialization with SDK 54.
